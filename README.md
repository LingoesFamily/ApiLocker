# API 钥匙扣

一个现代化的API密钥管理工具，帮助您安全地存储、组织和快速访问各种API凭证。

## ✨ 特性

- **安全存储**：API Key默认隐藏，保护敏感信息
- **多平台支持**：支持DeepSeek、智谱 AI、OpenAI、百度文心、阿里通义、高德地图等主流平台
- **智能分类**：通过分类和标签系统组织您的API凭证
- **快速搜索**：支持按平台、账号、ID、分类、标签等快速搜索
- **CSV导出**：一键导出所有API记录为CSV文件，方便备份和分析
- **响应式设计**：完美适配桌面和移动设备
- **本地存储**：数据保存在浏览器localStorage中，无需后端服务

## 📦 安装与使用

### 直接使用

1. 将 `1.2.html` 文件保存到您的本地
2. 用现代浏览器（Chrome、Firefox、Edge等）打开该文件
3. 开始管理您的API密钥

### 功能介绍

#### 添加API
点击"添加"按钮，填写以下信息：
- **平台**：选择或输入API平台名称
- **账号名称**：便于识别的账号标识
- **分类**：自定义分类，便于分组管理
- **标签**：用逗号分隔的标签，便于筛选
- **API ID**：API标识符
- **API Key**：API密钥（敏感信息）
- **管理后台链接**：快速跳转至对应平台的管理后台
- **备注**：记录用途、限额、到期时间等信息

#### 查看与操作
- **折叠/展开**：点击卡片右上角的折叠按钮，节省界面空间
- **显示/隐藏**：点击眼睛图标切换API Key的显示状态
- **复制**：点击复制按钮快速复制相应内容
- **编辑/删除**：通过卡片右上角的编辑和删除按钮管理记录

#### 搜索与筛选
- **搜索框**：支持多字段搜索
- **筛选器**：按平台、分类、标签进行筛选
- **重置**：一键清除所有筛选条件

#### 导出数据
点击"导出CSV"按钮，将当前筛选后的所有API记录导出为CSV文件。

## 🛠️ 技术栈

- **前端**：HTML5、CSS3、JavaScript (ES6+)
- **UI框架**：自定义CSS，响应式设计
- **图标库**：Font Awesome
- **数据存储**：浏览器localStorage

## 📝 注意事项

- 所有数据保存在浏览器的本地存储中，清除浏览器数据会导致数据丢失
- 建议定期导出CSV文件进行备份
- API Key默认隐藏，点击眼睛图标可切换显示状态
