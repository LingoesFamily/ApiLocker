帮助文档

📌 基本功能

  • 添加 API：点击「添加」按钮，填写平台、账号、分类、标签、API ID、API Key、管理后台链接和备注。
  • 编辑/删除：每个卡片右上角有编辑（✎）和删除（🗑）按钮。
  • 搜索：支持搜索平台、账号、API ID、分类、标签、链接地址。
  • 筛选：按平台、分类、标签进行筛选，可组合使用。

🔐 安全与隐私

  • API Key 隐藏：默认显示为掩码，点击眼睛图标可切换明文/密文。
  • 复制功能：每个字段旁的复制按钮可快速复制内容。

📁 卡片折叠

  • 每个卡片头部右侧有一个折叠按钮（⬇/⬆），点击可折叠/展开卡片主体，只保留标题区域（平台、账号、标签、操作按钮）。

📤 导出 CSV

  • 点击「导出CSV」按钮，将当前筛选后的所有 API 记录导出为 CSV 文件，方便数据分析或备份。

💡 小提示

  • 所有数据保存在浏览器的本地存储中，清除浏览器数据会导致丢失，建议定期导出备份。
  • 管理后台链接支持自定义，方便快速跳转至对应平台的控制台。
